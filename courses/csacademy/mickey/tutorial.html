Mickey Starter: Circles → Groups

Draw circles, style them, group them, then move the whole character.

1. Required Circle Arguments

Every Circle in CMU CS Academy needs three positional arguments in this order:

head = Circle(centerX, centerY, radius)
  • centerX: horizontal position (0 left → 400 right)
  • centerY: vertical position (0 top → 400 bottom)
  • radius: distance from center to edge in pixels

Example:

head = Circle(200, 240, 120)

2. Optional Style Arguments

Add keyword arguments after the required ones to customize colors and outlines:

  • fill="color"
  • border="color"
  • borderWidth=4
  • opacity=80
leftEar = Circle(105, 120, 70, fill="black")
rightEar = Circle(295, 120, 70, fill="black", border="gray", borderWidth=2)

3. Place Head + Ears

Create three circles to match the classic silhouette:

  1. Head centered lower on the canvas
  2. Left ear shifted up-left
  3. Right ear mirrored up-right
head = Circle(200, 240, 120, fill="black")
leftEar = Circle(105, 120, 70, fill="black")
rightEar = Circle(295, 120, 70, fill="black")

Sketching on paper first helps you estimate good center points.

4. Build a Group

Groups let you move multiple shapes together.

mickey = Group(leftEar, rightEar, head)

You can also start empty and add() shapes one at a time.

mickey = Group()
mickey.add(leftEar)
mickey.add(rightEar)
mickey.add(head)

5. Change Group Properties

Use dot notation to transform the entire character:

mickey.centerX = 200
mickey.centerY = 200
mickey.width *= 0.8
mickey.height *= 0.8
mickey.rotateAngle = -5

Later you can animate these properties inside onStep or onKeyHold.

Checklist

  • ✅ Three circles drawn with correct coordinates
  • ✅ Optional styling applied
  • ✅ Group created and populated
  • ✅ Group moved/scaled/rotated with dot notation
