CS Academy Mickey Starter

Circles ➜ Group ➜ Dot notation

Introduction

This tutorial helps you build the classic Mickey silhouette in CMU CS Academy by combining three circles into a single Group. You will learn the required arguments for Circle, how to add optional styling, position shapes precisely, and then transform the entire character with dot notation.

What you'll practice:
  • Using Circle(centerX, centerY, radius)
  • Applying optional keyword arguments like fill and borderWidth
  • Creating and populating Group objects
  • Changing Group properties (position, size, rotation)

Circle Essentials

Every circle starts with three required arguments:

head = Circle(centerX, centerY, radius)
  • centerX – horizontal position (0 on the left, 400 on the right)
  • centerY – vertical position (0 at the top, 400 at the bottom)
  • radius – size in pixels, measured from center to edge

Example head circle:

head = Circle(200, 240, 120)

Optional Style Arguments

After the required arguments, you can add keyword arguments to change colors and outlines.

fill="black"
border="gray"
borderWidth=3
opacity=80
leftEar = Circle(105, 120, 70, fill="black")
rightEar = Circle(295, 120, 70, fill="black", border="gray", borderWidth=2)

Keyword arguments always come after the positional ones.

Build the Silhouette

Create the head and two ears to match the Mickey outline.

  1. Place the head toward the lower center.
  2. Place the left ear above and to the left of the head.
  3. Mirror that position for the right ear.
head = Circle(200, 240, 120, fill="black")
leftEar = Circle(105, 120, 70, fill="black")
rightEar = Circle(295, 120, 70, fill="black")
Tip: Sketch coordinates on graph paper or use the CS Academy grid overlay to estimate positions quickly.

Create a Group

Groups let you move or resize multiple shapes together.

mickey = Group(leftEar, rightEar, head)

Or add shapes one at a time:

mickey = Group()
mickey.add(leftEar)
mickey.add(rightEar)
mickey.add(head)

Grouping ensures the ears follow the head when you transform the group later.

Transform with Dot Notation

Every Group exposes properties such as centerX, width, and rotateAngle. Changing them affects all shapes inside.

mickey.centerX = 200
mickey.centerY = 200
mickey.width *= 0.85
mickey.height *= 0.85
mickey.rotateAngle = -5

Later you can animate these values in onStep or respond to keyboard input in onKeyHold.

Checklist

  • ✅ Head + ears drawn using required circle arguments
  • ✅ Optional styling applied to at least one circle
  • ✅ All shapes added to a Group
  • ✅ Group moved/scaled/rotated with dot notation
