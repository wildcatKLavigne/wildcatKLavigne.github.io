Unit 1 of 9

Engineering Foundations

Students build the mental models and documentation habits that underpin all future engineering work: purposeful design, the Engineering Design Process, multi-view sketching and documentation, weighted design evaluation, and modular thinking. Skills compound across all five lessons and are assessed in the unit test.

Class Lessons
Class 1
What is Engineering? Purposeful Design
Discussion, robot photo analysis, and engineering notebook setup. Students examine visible design decisions in real robot photos using I Do / You Do modeling, then make their first formal notebook entry.
📋 Activity: Robot photo analysis · Notebook setup
Open worksheet Notebook template Worksheet .docx

Learning objectives

  • Define purposeful design and distinguish it from trial-and-error
  • Identify design decisions in a robot photo and explain the reasoning and trade-off
  • Set up an engineering notebook and make the first dated, signed entry
Novice: Guided analysis of 3 design choices from a provided photo
Intermediate: Compare two robots; write a comparison paragraph
Proficient: Write a design critique; propose one improvement with full rationale
Class 2
The Engineering Design Process
Introduce the 6-stage EDP loop. Students apply the full process to a constrained physical challenge — three timed iterations with data collection and root-cause analysis between each attempt.
🏗️ Activity: Paper tower challenge — 3 sheets, 3 iterations
Open worksheet Notebook template Worksheet .docx

Learning objectives

  • Name and describe all 6 EDP stages in order
  • Apply EDP to a physical challenge across 3 timed iterations with data-driven changes
  • Explain why iteration is a designed feature of engineering, not a sign of failure
Novice: Fill in EDP organizer; document attempt results with guided prompts
Intermediate: Write root-cause analysis; calculate improvement percentage
Proficient: Predict Attempt 3 height from A1→A2 trend; write formal iteration log
Class 3
Engineering Documentation & Sketching
Orthographic views, dimension labeling, title blocks, and revision boxes. Students sketch three designs for each of two objectives, evaluate with decision matrices, and iterate across three builds per objective.
✈️ Activity: Paper airplane challenge — distance & accuracy, 3 iterations each
Open worksheet Notebook template Worksheet .docx

Learning objectives

  • Sketch in orthographic views with dimension labels, title block, and revision box
  • Generate 3 designs per objective using sketching standards; evaluate with a decision matrix
  • Apply multi-objective design thinking: distance vs. accuracy trade-offs
Novice: Sketch two views with templates; fill in decision matrix using provided criteria
Intermediate: Freehand all views; calculate improvement %; explain trade-off paragraph
Proficient: Full orthographic documentation; annotate force path; Challenge C combined-score analysis
Class 4
Engineering Documentation & the REV Servo Claw
Introduces the weighted decision matrix. Students design a REV Smart Robot Servo-powered claw mechanism using the full EDP: define requirements, research mechanisms, sketch three designs, evaluate with weights, and produce a final engineering drawing with parts list.
🦾 Activity: REV servo claw design — full orthographic drawing + parts list
Open worksheet Notebook template Worksheet .docx

Learning objectives

  • Use a weighted decision matrix: assign weights before sketching, multiply ratings, total scores
  • Sketch 3 claw mechanism designs in orthographic views; mark servo position and mounting interface
  • Produce a final 3-view engineering drawing with title block, revision box, and parts list
Novice: Vary finger shapes across 3 designs; complete matrix with provided criteria
Intermediate: 3 genuinely different mechanism types; complete weighted matrix; write rationale
Proficient: Annotate force path; calculate torque at fingertip; GD&T extension
Class 5
Modular Design Thinking
Introduces the five properties of modular design. Students build the LEGO® Technic™ 42197 Backhoe Loader, identify its four modules, document each with orthographic sketches, draw the architecture diagram, evaluate the model's modularity, and redesign one module with a weighted decision matrix.
🔩 Activity: LEGO® Technic™ 42197 Backhoe Loader — build, document, redesign
Open worksheet Notebook template Worksheet .docx

Learning objectives

  • Define modular design and name its five properties: functional independence, defined interfaces, replaceability, testability, parallel development
  • Identify modules and interfaces in the 42197; document each module with orthographic sketches
  • Evaluate a design's modularity with a scoring rubric; propose a redesign with 3 alternatives
Novice: Label 4 modules; fill in module cards with teacher support
Intermediate: Draw architecture diagram; evaluate 2 modular properties with evidence
Proficient: Full interface map; knob-to-arm power transmission analysis; ISS modularity extension
Assessment
Unit 1 Test — Engineering Foundations
100 points · Answer bank (Section 1) · Free response · Two performance tasks

Covers all five classes. Section 1 is an answer-bank format (10 questions, 2 pts each). Section 2 has three free-response questions on purposeful design, EDP, and modular design. Section 3 has two performance tasks: a motor mount design with weighted evaluation and a modular robot architecture.
